# Docker（轻量级的虚拟容器）

- http://www.runoob.com/docker/docker-tutorial.html

- 什么是Docker

```
1 Docker 是支持将软件编译成一个镜像，在这个镜像里做好了对各种软件的各种配置。
2 运行中的镜像称为容器
 
```

- Docker 常用命令以及参数


  |命令|解释|
  | :--------  | :--------  |
  |docker search 镜像名|镜像的检索|
  |docker pull 镜像名 |下载镜像|
  |docker images|镜像列表|
  |docker run -d -p 6378:6379 redis|-d:控制台将不会被阻碍  -p端口映射|
